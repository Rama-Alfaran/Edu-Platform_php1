# 🎓 Edu Platform (PHP)

A simple educational platform that connects **students and instructors**, allowing easy course management and participation.

---

## 📌 Project Overview

This project is a lightweight web application built with **PHP** that enables:

- Instructors to create and manage courses  
- Students to browse and join available courses  
- Basic interaction between students and instructors  

---

## 👨‍🏫 Instructor Features

- Create and manage courses  
- View enrolled students  
- Dashboard for managing content  
- Send messages to students  

---

## 👩‍🎓 Student Features

- View available courses  
- Enroll in courses  
- Access their registered courses  

---

## 🛠️ Technologies Used

- PHP  
- MySQL  
- HTML / CSS  
- Composer  

---
